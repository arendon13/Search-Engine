Hp 620lx

From Wikipedia, the free encyclopedia

Hp 620lx
Type Palmtop
Developer Hewlett Packard
Processor 75MHz Hitachi SH3
Memory 16 MB
Display 640x240 pixels
Input Keyboard, Touchscreen
Connectivity CompactFlash type I
Operating system Windows CE 2.0

The Hp 620lx is a handheld, palmtop organizer that runs Windows CE 2.0. It has a Compact-flash type 1 card slot, a PC card slot, a serial link cable plug, and an Infrared Port.

It is internet capable by attaching an add-on modem or through a ethernet or wifi card. Only type 1 PC cards are supported and you need special drivers made for the Windows CE operating system.
