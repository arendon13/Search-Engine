List of XInput games

From Wikipedia, the free encyclopedia

This is a list of Windows games that require the XInput library from Microsoft; games that may optionally load the library should not be included on this list. XInput is used to provide native support for the Xbox 360 controller and other compatible controllers in Windows. Generally, XInput only appears with multiplatform games, presumably to simplify concurrent development of the PC and Xbox versions. The XInput package does not install to versions of Windows earlier than Windows XP with Service Pack 1.[1] This is of special interest to users of Windows 2000 and base versions of Windows XP because these operating systems are usually able to run modern Windows games that do not explicitly support them. The requirement of XInput, however, prevents games from running on these systems, regardless of whether an Xbox controller is being used. In some cases, if XInput is the only library preventing the game from running, it is possible to work around the issue by manually copying the dll into the system32 folder. The dll, which has a filename of the form xinput_1_n.dll, can be manually extracted from the DirectX CAB file found on the game disc.
